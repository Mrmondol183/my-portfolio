# 🚀 CSE Graduate Portfolio

A modern, responsive portfolio website for Computer Science graduates to showcase skills, projects, and professional experience.

## ✨ Features

- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Smooth Animations** - Scroll animations and hover effects
- **Dynamic Content** - JSON-based project management
- **Contact Form** - Working UI with validation
- **Project Gallery** - Showcase your best work
- **Skills Section** - Display technical competencies
- **Modern UI** - Clean design with gradient hero section

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- JSON for dynamic content
